Introduction

What you will learn

During the course of this tutorial, you will learn the following:

  • How to provide client credentials to authenticate your app
  • How to let your users log in to Creative Cloud with your app
  • How to access metadata related to a photo selected by your user from Creative Cloud

What you should already know

The following basic knowledge is recommended for this tutorial:

  • Some basic experience with Android Studio and Android Studio Virtual Devices
  • Familiarity with Java

Tools you'll need

We'll be using the following tools to build our Android app with Adobe Creative SDK.

Android development tools

  • Android Studio version 1.2.2 or higher
  • Android SDKs

    To manage your installed SDKs for Android, in the Android Studio menu bar go to:

    Tools > SDK > SDK Manager

    Be sure to install these minimum requirements:

    1. All tools in the "Tools" folder
    2. All SDKs in the "Android 5.1.1 (API 22)" folder
    3. All SDKs in the "Android 4.0.3 (API 15)" folder
    4. The following SDKs in the "Extras" folder:
      • Android Support Repository
      • Android Support Library
      • Google Play Services
      • Google Repository

Adobe Creative SDK

  • A downloaded copy of Creative SDK for Android version 0.4.264 or higher
  • A new app registered under "My Apps" for Adobe Creative SDK
  • The Client ID and Client Secret for your newly registered app
